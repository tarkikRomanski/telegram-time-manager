## Overview

Ran `npx fallow` static analysis on the project, identified several dead code, dependency, duplication, and complexity issues, and implemented comprehensive refactorings to resolve all findings.

Key improvements include:
- **0% Dead Code / Unused Exports**: Cleaned up `components/formation.js` by exporting only actively consumed functions (`bold`).
- **Dependencies Restructured**: Moved `dotenv` to `dependencies` in `package.json` for proper production usage and removed 6 unused devDependencies.
- **Zero Duplication**: Extracted repeated task query logic and Telegram message option creation into a shared helper module `endpoints/helpers.js`.
- **Complexity Reduced**: Streamlined `index.js` message dispatch using a command route table, reducing cognitive complexity from 30 down to 9 and CRAP risk score from 870.0 to 90.0.
</RichText>
